        that a menu is present.<br /><br />
        <Image src={closedListMenuButton} alt="image" style="width: 55px"/>

<div class="topic">
  Ctrl-Tab
  <ul>
    <li>
      Ctrl-Tab navigates tabs in the most recent used order (pressing Ctrl-Tab will navigate
      between Tabs in the most recently used order)
      <ul class="square">
        <li>Show tab previews - displays a window with tab previews.</li>
        <p class="comment full-width">
          This preference takes precedence over the next preference.
        </p>
      </ul>
    </li>
    <li>
      Ctrl-Tab displays a tab list popup menu - pressing Ctrl-Tab will display a popup menu
      list of all tabs.
      <ul class="square">
        <li>
          Tab list responds to the mouse - when checked you can use your mouse left click on
          the list to select a tab
        </li>
      </ul>
      <p class="comment full-width">
        The List will sort tabs in most recently used order if you turn on
        <b>Ctrl-Tab navigates tabs in the most recent used order</b>.
      </p>
    </li>
  </ul>
</div>

<div class="topic">
  <ul>
    <li>
      Toggle tabs slideshow - click to go to Menu/Shortcut to enable/disable or change the
      shortcut
    </li>
    <li>Disable Open/Close tab animation (removes default Firefox animation from tabs)</li>
  </ul>
</div>

<div class="topic">
  Auto Reload
  <ul>
    <li>
      Reload a tab regardless of its address - the tab will be reloaded even if its URL
      changes
    </li>
    <li>
      Show Reload Every menu on Reload button - a Reload every menu will be added to the
      Reload toolbar button
    </li>
  </ul>
</div>
