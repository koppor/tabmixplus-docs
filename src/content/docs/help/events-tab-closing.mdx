#####  Controls tab closing, when warnings will appear and where to set focus when a tab is closed

<br />

<Image class="help-img" src={eventsTabClosing} alt="image"/>

<div class="topic">
  Warning:
  <ul class="list-disc pl-8">
    <li class="default">Warn me when closing multiple tabs</li>
    <li class="default">Warn me when closing window with multiple tabs</li>
    <li class="default">Warn me when closing window with protected tabs</li>
  </ul>
</div>

<div class="topic">
  Closing last tab:
  <ul class="list-disc pl-8">
    <li>
      Do not close window when closing the last tab (closing the last tab will also close the
      browser window)
    </li>
    <li>Prevent last tab from closing</li>
  </ul>
</div>

<div class="topic">
  <ul class="list-disc pl-8">
    <li>
      When I close the last tab replace it with (select one):
      <ul class="list-disc pl-8">
        <li>
          <span class="default">New Tab Page</span> - see more details on
          <a href="http://support.mozilla.org/en-US/kb/new-tab-page-show-hide-and-customize-top-sites">Firefox Help</a>
        </li>

        <li>Blank Page</li>
        <li>
          Home Page (to select your Home Page you must specify a Home Page in the Firefox
          Options/General)
        </li>
        <li>Current Page (whatever page is currently displayed in your browser)</li>
        <li>Duplicate Tab with history (copy the current tab with history)</li>
      </ul>
    </li>
    <li>
      Focus content when loading non blank page, (when not checked - focus is on the address
      bar)
    </li>
  </ul>
</div>

<div class="topic">
  When closing the current tab, focus (select one):
  <li style="list-style-type: none; margin-left: 0px">
    <ul class="circle">
      <li>First tab</li>
      <li>Left tab</li>
      <li>Right tab</li>
      <li>Last tab</li>
      <li>Last selected tab</li>
      <li>
        <span class="default">Opener/Right tab</span> (the tab containing the link that opened
        this tab or the tab to the right)
      </li>
      <li>Last opened tab</li>
    </ul>
  </li>
</div>
