#### Customize Tab Styles
<div class="topic">
  <a name="customize-styles"></a>
  Click on <b>Customize Styles</b> button to open Customize Styles editor in a separate
  dialog. The dialog contains five panels: Current tab, Unread tabs, Unloaded tabs, Other tabs
  and Progress meter on tabs.
</div>

<Image class="help-img" src={displayTabStyles} alt="image" />

<div class="topic">
  <ul class="list-disc pl-8">
    <li>
      Use this style - enable/disable spesific style, same as changing one of the appropriate
      options (listed above) in the main dialog window
    </li>
    <li>Italic</li>
    <li>Bold</li>
    <li>Underline</li>
    <li>Text Color: select color and specify opacity `(100%)`</li>
    <li>Background Color: select color and specify opacity `(100%)`</li>
    <li>Restore Defaults (resets spesific style to default)</li>
  </ul>
  <aside aria-label="Note" class="starlight-aside starlight-aside--note">
    You can control tab title bold, italic and underline style as well as the title color
    and opacity. For the tab background, you can control color and opacity, set different
    colors and opacity on tab bottom and top to get nice gradient effect of the color.<br />
    Each color control has five parts:
    <ul class="list-disc pl-8">
      <li>button with predefined colors</li>
      <li>three fields for user-defined color (color range 0-255)</li>
      <li>opacity field for defined the opacity level of the color (range 0%-100%)</li>
    </ul>
  </aside>
</div>
