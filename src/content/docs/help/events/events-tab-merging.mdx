##### Controls tabs and windows merging when tabs are closed

<br />

<Image class="help-img" src={eventsTabMerging} alt="image" />

<div class="topic">
  Merge windows when no tabs are selected (Select one option):
  <ul class="list-disc pl-8">
    <li class="default">Merge all windows into one</li>
    <li>Merge only current window with last focused</li>
    <li>
      Also take popups - popup windows will also be merged
      <ul class="square">
        <li>
          Place popups next to their openers - popups will be placed next to their openers
        </li>
      </ul>
    </li>
  </ul>
</div>

<div class="topic">
  Merging after selecting tabs
  <ul class="list-disc pl-8">
    <li>
      <span class="default">Close windows once they have merged</span>
      (when all tabs are merged from other windows, those windows will be closed)
      <ul class="square">
        <li>
          <span class="default">Warn when closing tabs that aren't being merged</span> - a
          warning will be displayed indicating not all tabs have been merged
        </li>
      </ul>
    </li>
  </ul>
</div>
