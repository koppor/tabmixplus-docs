<div class="topic">
  <h6>Ctrl-Tab</h6>
  <ul class="list-disc pl-8">
    <li>
      Ctrl-Tab navigates tabs in the most recent used order (pressing Ctrl-Tab will navigate between
      Tabs in the most recently used order)
      <ul class="square list-disc pl-8">
        <li>Show tab previews - displays a window with tab previews.</li>
        <aside class="starlight-aside starlight-aside--note">
          This preference takes precedence over the next preference.
        </aside>
      </ul>
    </li>
    <li>
      Ctrl-Tab displays a tab list popup menu - pressing Ctrl-Tab will display a popup menu list of
      all tabs.
      <ul class="square list-disc pl-8">
        <li>
          Tab list responds to the mouse - when checked you can use your mouse left click on the
          list to select a tab
        </li>
      </ul>
      <aside class="starlight-aside starlight-aside--note">
        The List will sort tabs in most recently used order if you turn on
        <b>Ctrl-Tab navigates tabs in the most recent used order</b>.
      </aside>
    </li>
  </ul>
</div>
