<div class="topic">
  <u><b>Protected tab</b></u>
  <p>
    A tab that is prevented from closing. You can load other sites in the tab and browse to
    other URLs in that tab. Notice that the close button is missing, so you cannot close the
    tab. If you try to close Firefox&reg;, you will get a warning message about closing a
    protected tab.
  </p>

  <u><b>Locked tab</b></u>
  <p>
    A tab that is prevented from navigating to another page or URL. Your tab is locked on the
    current page or URL. Nothing new can load in the tab, but you can refresh the same page.
    All links will open in a new tab (link, history, bookmark, etc.). Notice that the close
    button is available, so you can close a locked tab.
  </p>

  <u><b>Frozen tab</b></u>
  <p>
    A tab that is both locked and protected. It will not close and nothing new can be loaded
    in it. You will not be able to navigate away from the current page. All links will open in
    a new tab (link, history, bookmark, etc.) Notice that the close button is missing, so you
    cannot close the tab. If you try to close Firefox&reg;, you will get a warning message
    about closing a protected tab.
  </p>
</div>

<div class="grid xl:grid-cols-2 gap-2">
  <div class="topic container">
    <u>Lock New tabs</u>
    <p>
      You can set Tab Mix Plus to lock every new tab and every new App tab. If "Apply changes to
      open tabs" is enabled, any change to <b>Lock New tabs</b> or
      <b>Lock Apps tabs</b> preferences changes the lock status of open tabs accordingly.
    </p>
  </div>
  <Image class="w-10/12" src={lockNewTabs} alt="image" />
</div>

<div class="topic container">
  <u>Change single tab Protected, Locked or Frozen status</u>
  <p>
    You can change single tab Protected, Locked or Frozen status from tab context menu or
    browser main context menu. To enable these options in the context menu, go to Tab Mix Plus
    Option window > Menu Panel and set the options you need for each menu.
  </p>
</div>
