##### Controls keyboard shortcuts

<br />

<Image class="help-img" src={shortcuts} alt="image" />

Edit shortcuts by selecting one and type Modifiers & Key <br /> while in the edit box, you can use
the button on the right side to restore shortcuts to default (if exist) or clear/disable the
shortcuts. <br /> You can also use Delete or Backspace to clear the shortcut
